# kerta

Repository ini berisi script yang digunakan dalam pengembangan dan pengujian model keterbacaan kode Kerta. Model fitur Kerta terbentuk dari 57 fitur metrik yang diekstraksi dari program dan bersifat umum untuk semua bahasa pemrograman berbasis objek. Untuk saat ini pengembangan model Kerta masih fokus pada cuplikan kode Java sebagai *proof of concept*.
